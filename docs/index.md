# Internet 架构与协议简介

_作者：[Peyrin Kao](https://peyrin.github.io)，基于 [Sylvia Ratnasamy](https://www2.eecs.berkeley.edu/Faculty/Homepages/ratnasamy.html)、[Rob Shakir](https://rob.sh/) 等人的课程讲义。_

这是 [UC Berkeley](https://eecs.berkeley.edu/) [CS 168: Introduction to the Internet](https://cs168.io/) 的课程讲义。

以下是官方课程描述：

<p class="blue">
这门课介绍 Internet architecture。我们会关注支撑 Internet 可扩展性与稳健性的概念和基本设计原则，并概览这一架构中使用的各种 protocol 与 algorithm。主题包括 layering、addressing、intradomain routing、interdomain routing、reliable delivery、congestion control，以及核心 protocol（例如 TCP、UDP、IP、DNS 和 HTTP）和网络技术（例如 Ethernet、wireless）。
</p>
