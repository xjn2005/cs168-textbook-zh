# CS 168 Textbook 中文翻译

本仓库为 UC Berkeley CS 168 Textbook 的中文翻译项目。原项目：[cs168-textbook](https://textbook.cs168.io/)

翻译过程中尽量保持原意，兼顾可读性，希望能为中文读者提供一份高质量的计算机网络学习资料。

在线阅读：[Textbook](https://xjn2005.github.io/cs168-textbook-zh/)

## 本地运行

构建：

```powershell
npm run docs:build
```

预览：

```powershell
npm run docs:serve
```
